**Fantasy Map Generator**

**What This Tool Does**
This Map Generator generates a fantasy world using layered noise fields.
Users can control the terrain size, climate, island count, & city density.

**How To Run**
https://bkkmwqgvgjp7zupas8hvl5.streamlit.app/

**Parameters**
- World Size & Resoluaiton of the Map
- Terrain Size
- Island Fragmentation
- Temperature
- Number of Cities
- Seed

**Known Limitations**
- No Rivers
- Cities don't consider the biome they're in.
- Biomes have straight borders
